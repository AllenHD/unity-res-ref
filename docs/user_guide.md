# User Guide

## Installation

```bash
pip install unity-res-ref
```

## Quick Start

1. Initialize configuration:
```bash
unity-res-ref init /path/to/unity/project
```

2. Scan project:
```bash
unity-res-ref scan
```

3. Find dependencies:
```bash
unity-res-ref find-deps Assets/MyAsset.prefab
```

4. Export results:
```bash
unity-res-ref export --format json
```
